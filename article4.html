Project 4

Federated Continual Learning for Adaptive DDoS Detection in Heterogeneous IoT Environments

Type: AI, Federated Learning, Continual Learning, Network Security, IoT
Publication Date: June 2025

This project introduces a Federated Continual Learning (FCL) framework aimed at enhancing the detection of Distributed Denial-of-Service (DDoS) attacks in heterogeneous Internet of Things (IoT) environments. The framework tackles key challenges such as data privacy, non-IID data distribution, and the evolving nature of cyber threats by combining continual learning with federated training strategies.

The proposed approach utilizes advanced continual learning techniques—Prioritized Experience Replay (PER), Learning Without Forgetting (LWF), and Elastic Weight Consolidation (EWC)—to mitigate catastrophic forgetting. These techniques enable models deployed on IoT gateways to retain past knowledge while adapting to new threats using minimal memory and computational overhead.

Federated learning strategies—FedAvg, FedProx, AMU, and CFL—are integrated to enable secure and decentralized model updates from multiple clients. Experiments conducted with 16, 32, and 64 active clients using the CIC-IoT2023 dataset demonstrated that FedProx and FedAvg consistently outperformed other methods in terms of accuracy, precision, and recall, especially when combined with LWF and EWC.

The deep learning model architecture is inspired by ResNet and tailored for one-dimensional network stream classification. It features residual blocks, batch normalization, dropout layers, and three fully connected layers, ultimately classifying network traffic into 32 distinct classes.

As a key contributor, I was involved in designing and implementing the FCL framework, conducting extensive experimentation, and analyzing the performance of different continual and federated learning strategies. I also optimized training pipelines on PyTorch, processed real-world IoT DDoS data, and applied CL-based mitigation to reduce retraining overhead.

In conclusion, this project delivers a scalable and adaptive intrusion detection solution by combining edge-based continual learning and federated optimization. Future work will explore dynamic client participation, energy-efficient client coordination, and incremental learning of novel attack types without retraining from scratch.
